# CULTUS-PROJECT-NM
This project forecasts hourly electricity demand using the HAR-ARIMA model with PJM data. The model captures hourly, daily, and weekly patterns using three features: Lag-1, 24-hour mean, and 168-hour mean.Compared with SARIMA and XGBoost, HAR-ARIMA provides strongaccuracy,faster training,and clear interpretability for real-world energy forecasting.
